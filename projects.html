Founder & Distributed Systems Lead. A social platform backed by 12 bounded Go microservices with decoupled REST/gRPC contracts, geospatial + vector recommendations, and a cross-platform React Native client.

Go Go-Chi gRPC PostgreSQL PostGIS / H3 FAISS / pgvector React Native TypeScript
12 Microservices Independently deployable, versioned boundaries
Geo + Vector Search PostGIS, Geohash, H3, FAISS, pgvector
Privacy by Design Block/report semantics on every read path

Architected the backend as 12 bounded microservices (Auth, Social Graph, Feed, Plans, Notifications, and more) in Go using Go-Chi, with decoupled REST/gRPC contracts for independent deployability and versioned service boundaries. Designed relational data models in PostgreSQL backing the friend graph and plan lifecycle, enforcing cross-cutting privacy constraints and block/report semantics on every read path.

Integrated geospatial indexing (PostGIS, Geohash, H3) and vector similarity search (FAISS, pgvector) to power low-latency location- and interest-based recommendation pipelines, and built a cross-platform React Native/TypeScript client with OAuth (Google/Apple), JWT refresh-token rotation, and device-scoped push notifications. Led a cross-functional engineering, design, and growth team through agile sprints, code review, and release cycles.
